## 권장 이미지 사양

- 형식: JPG 또는 PNG
- 최소 해상도: 400x300px (4:3 비율)
- 최대 파일 크기: 1MB 이하
- 최적화된 웹용 이미지 권장

이미지가 없을 경우 자동으로 fallback 디자인이 표시됩니다.
