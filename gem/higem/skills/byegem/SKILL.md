# byegem — 세션 마무리 저장 트리거 (da da-x 협업)

## 목적

세션 전체 대화를 두 곳에 저장 가능한 형태로 정리한다:
- **NotebookLM** — 세션 로그 (출력물 A)
- **Google Docs 브리프** — 다음 세션용 업데이트 제안 (출력물 B)

두 출력물은 용도가 다르므로 **반드시 분리해서** 제공한다.

## 발동 조건

- 사용자 메시지가 정확히 `byegem` (대소문자 무관, 구두점 무관)
- 사용자가 명시적으로 "세션 마무리", "저장 정리해줘" 등 종료 요청

## 출력물 A — NotebookLM 저장용 (세션 로그)

다음 양식을 그대로 출력한다 (사장님이 NotebookLM에 그대로 붙여넣을 수 있어야 한다):

```
# da da-x 세션 로그 — [YYYY-MM-DD]

## 다룬 주제
- (오늘 대화의 큰 흐름 3~5줄)

## 결정된 사항
- (확정되어 향후 행동 지침이 되는 항목들. 추측이나 미확정 사항은 절대 여기 넣지 않는다)

## 보류된 질문
- (논의했으나 아직 결정되지 않은 항목들)

## 다음 액션
- (다음 세션에서 이어서 할 구체적 작업들)

## 핵심 인사이트
- (이번 세션에서 새롭게 명확해진 통찰. 1~3개)
```

**날짜는 오늘 날짜를 자동으로 채워 넣는다** (env에서 제공된 currentDate 사용).

## 출력물 B — Google Docs 브리프 업데이트 제안

```
## 브리프 업데이트 제안 — v[다음 버전]

**8번 항목(현재 진행 상태) 변경안**:
- (수정해야 할 내용)

**10번 항목(다음 업데이트 예정) 변경안**:
- (추가/삭제/수정할 내용)

**기타 항목 변경안** (있을 경우):
- (어느 섹션의 어떤 부분을 어떻게 바꿀지)
```

**버전 표기**: 사장님이 현재 브리프 버전을 알려줬다면 +1, 아니면 `v[다음]`처럼 빈 자리로 둔다.

## 두 출력물 분리 표시

A와 B 사이에는 시각적으로 명확한 구분이 필요하다. 다음 형태를 권장:

```
========================================
📋 출력물 A — NotebookLM 저장용
========================================

[A 본문]

========================================
📋 출력물 B — Google Docs 브리프 업데이트 제안
========================================

[B 본문]
```

(이모지 사용은 사장님이 명시적으로 선호하지 않는 한 텍스트 구분선만 사용해도 좋다. `---` 굵게 또는 `===`로 충분.)

## 핵심 원칙 (반드시 지킬 것)

1. **결정사항과 보류된 질문은 절대 섞지 않는다.**
   - "결정된 사항": 사장님이 명시적으로 OK했거나, 작업 결과가 확정된 것만.
   - "보류된 질문": 논의는 했으나 결론이 안 난 것. "~할까?", "~인 것 같은데"는 모두 보류.

2. **추측 금지.** 대화에서 직접적으로 도출되지 않은 항목은 절대 넣지 않는다. 빈 섹션이 있으면 그대로 비워두거나 "(이번 세션 해당 없음)"으로 표기.

3. **A는 NotebookLM 호환 마크다운**, B는 Google Docs에 옮길 수 있는 텍스트로 작성. 코드블록(```)으로 감싸지 말고 마크다운 그대로 출력 (그래야 복붙이 깔끔하다).

4. **간결성**: 각 bullet은 한 줄~두 줄. 장황한 설명·서사 금지.

## 절대 금지사항

- **A와 B를 합치거나 순서 바꾸지 않는다.**
- **"수고하셨습니다", "오늘도 좋은 하루..." 같은 마무리 멘트 금지.** 출력물만 깔끔하게 내고 끝.
- **이전 세션 내용을 byegem 출력에 끌어오지 않는다.** byegem은 *이번 세션*만 정리한다.
- **결정 안 된 것을 "결정된 사항"에 넣지 않는다.** 의심스러우면 "보류된 질문"으로.

## 후속 흐름

byegem 출력 후 세션은 사실상 종료된다. 사장님이 추가 요청이 없으면 더 이상 출력하지 않는다. 다음 세션은 `higem`으로 다시 시작된다.
